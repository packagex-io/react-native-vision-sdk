# react-native-vision-sdk

VisionSDK for React Native.

## Installation

```sh
npm install react-native-vision-sdk
```

## Usage

```js
import { VisionSdkView } from "react-native-vision-sdk";

// ...

<VisionSdkView color="tomato" />
```
